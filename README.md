# Movement Alert

#### Play Store Link: https://play.google.com/store/apps/developer?id=Mandar+Shinde

This utility helps you to get alarm when some one grabs your mobile phone without your permission. You just have to set mobile to any stable position like table or your work desk and start the app. Alarm will stop when mobile is set back to its original position.
